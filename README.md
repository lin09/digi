#### 安装
```
yarn add -D digi
```

----
#### 实例
- [demo](https://github.com/lin09/demo/tree/master/digi)

#### 简单例子
```
import digi, { createData } from 'digi'

// 创建监听数据
const data = createData()

// 添加元素
digi({ text: data.$tp('a') })

console.log(document.body.lastElementChild.outerHTML)
// => <div></div>

data.a = 321
console.log(document.body.lastElementChild.outerHTML)
// => <div>321</div>
```
